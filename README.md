## Getting Started

### Prerequisites

All you need is:
- A Google account
- A web browser
- Internet connection

### How to Run

1. Clone or fork this repository:
```bash
git clone https://github.com/Rishi5105-kashyap/Codtech-tasks.git
```

2. Navigate to a notebook file (`.ipynb`)

3. Click the "Open in Colab" button, or manually open it at:
```
https://colab.research.google.com/github/Rishi5105-kashyap/Codtech-tasks/blob/main/[notebook-name].ipynb
```

4. Run the cells sequentially to see the results

## Tips for Using Google Colab

- **Run All Cells**: Use `Runtime → Run all` to execute the entire notebook
- **Restart Runtime**: Use `Runtime → Restart runtime` if you encounter issues
- **GPU Access**: Enable GPU via `Runtime → Change runtime type → GPU`
- **Upload Data**: Use `files.upload()` to upload data files
- **Download Results**: Use `files.download()` to save outputs
- **Save to Drive**: Click the Drive icon to save notebooks to Google Drive

## Keyboard Shortcuts

- `Ctrl+M+B` - Add new cell below
- `Ctrl+M+A` - Add new cell above
- `Ctrl+Enter` - Run current cell
- `Shift+Enter` - Run current cell and move to next
